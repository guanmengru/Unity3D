# Unity3D
角色扮演游戏
场景为unity商城的成品
移动：WASD
跳跃：空格 
鼠标左键攻击，移动时按鼠标右键疾跑
esc返回主菜单
小门可以回血并存档
大门具备小门功能的同时，按下F会进行传送，出生点处的大门可以传送到第二场景，其他大门只能同场景传送
初次配置时可能会有报错，clear掉
File may be corrupted or was serialized with a newer version of Unity.这个报错在原版是不存在的，不知道为啥git版会有，但是不影响运行，clear
如果BOXOPHOBIC文件夹在代码编辑器中报错，不用管，不会影响游戏的运行，这只是unity商城中的天空盒子，我也不知道为什么报错，实在觉得恶心就把主菜单的天空盒子换了并删除BOXOPHOBIC文件夹
